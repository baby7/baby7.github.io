Java程序解析yml文件报错解决

发表于 2019-11-14 更新于 2024-08-28 分类于 后端技术

java.nio.charset.MalformedInputException: Input length = 1

Java程序解析yml文件报错解决

bug关键字：java.nio.charset.MalformedInputException: Input length = 1

过程：

运行报错说解析yml文件出现问题，不能正常解析，这个项目是我用Git Clone下来的，在别的电脑上都能跑的通，hosts文件配置也没有问题，但是就是这台电脑上运行不了
经过求助大佬，经过很长时间的找bug，最后发现是idea的setting中的项目编码有问题，我的IDEA以前设置过，忘了改回来，现在是GBK，应该改成UTF-8

解决：

File –> Settings –> Editor —>File Encodings ，将所有的格式都转成utf-8格式
